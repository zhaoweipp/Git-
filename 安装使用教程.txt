1:下载git scm
  https://git-scm.com/downloads

2:安装
  默认下一步一路点到底

3:使用
  打开git bash
  输入  ssh-keygen -t rsa -C'git邮箱' (网络上很多文章都是ssh -keygen，中间有空格，会导致Bad escape character 'ygen'.所以ssh-keygen中间是没有空格的)

4:克隆远程项目到本地
  git clone 地址 D:/xxx/xxx  直接clone一个远程项目到D:/xxx/xxx
  
5:修改本地项目

6：push项目到远程仓库
